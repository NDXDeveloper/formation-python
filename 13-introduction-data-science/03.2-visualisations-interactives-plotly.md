# 13.3.2 Visualisations interactives avec Plotly

## Introduction

Plotly est une biblioth√®que de visualisation moderne qui permet de cr√©er des graphiques **interactifs** de qualit√© professionnelle. Contrairement √† Matplotlib qui g√©n√®re des graphiques statiques, Plotly permet √† l'utilisateur d'interagir avec les donn√©es en temps r√©el : zoom, survol pour voir les valeurs, filtrage, etc.

**Pourquoi utiliser Plotly ?**
- **Interactivit√©** : zoom, pan, s√©lection, tooltips automatiques
- **Qualit√© professionnelle** : graphiques pr√™ts pour des pr√©sentations et dashboards
- **Facilit√© d'utilisation** : syntaxe intuitive
- **Export facile** : HTML, PNG, SVG, PDF
- **Parfait pour le web** : int√©gration simple dans des applications web

## Installation

Plotly s'installe facilement avec pip :

```bash
pip install plotly
```

Pour certains exports et fonctionnalit√©s avanc√©es :

```bash
pip install kaleido  # Pour exporter en images statiques
```

## Les deux interfaces de Plotly

Plotly propose deux fa√ßons de cr√©er des graphiques :

1. **Plotly Express** (recommand√© pour d√©buter) : interface simple et rapide
2. **Plotly Graph Objects** : interface plus d√©taill√©e pour un contr√¥le total

Nous allons principalement utiliser **Plotly Express** car il est plus simple et couvre la majorit√© des besoins.

## 1. Graphique en ligne interactif

### Exemple simple avec Plotly Express

```python
import plotly.express as px
import numpy as np

# Donn√©es
x = np.linspace(0, 10, 100)
y = np.sin(x)

# Cr√©ation du graphique
fig = px.line(x=x, y=y,
              title='Fonction sinus',
              labels={'x': 'Angle (radians)', 'y': 'sin(x)'})
fig.show()
```

**Fonctionnalit√©s interactives automatiques :**
- Survolez la courbe pour voir les valeurs exactes
- Zoomez avec la molette ou en s√©lectionnant une zone
- D√©placez le graphique (pan)
- Double-clic pour r√©initialiser le zoom

### Graphique multi-lignes

```python
import plotly.express as px
import pandas as pd
import numpy as np

# Cr√©ation d'un DataFrame avec plusieurs s√©ries

df = pd.DataFrame({
    'x': x,
    'sin': np.sin(x),
    'cos': np.cos(x),
    'tan': np.tan(x)
})

# Transformation pour Plotly (format long)
df_long = df.melt(id_vars='x', var_name='fonction', value_name='valeur')

# Cr√©ation du graphique
fig = px.line(df_long, x='x', y='valeur', color='fonction',
              title='Fonctions trigonom√©triques',
              labels={'valeur': 'f(x)', 'x': 'Angle (radians)'})
fig.show()
```

### Graphique avec markers (points)


## 5. Box plots (Bo√Ætes √† moustaches)

## Exemple complet : Dashboard interactif

Voici un exemple complet combinant plusieurs graphiques dans un dashboard :

```python
import plotly.graph_objects as go
from plotly.subplots import make_subplots
import pandas as pd
import numpy as np

# G√©n√©ration de donn√©es fictives
np.random.seed(42)
dates = pd.date_range('2024-01-01', periods=12, freq='M')
df = pd.DataFrame({
    'mois': [d.strftime('%b') for d in dates],
    'ventes': np.random.randint(50, 150, 12),
    'objectif': [100] * 12,
    'region_nord': np.random.randint(20, 60, 12),
    'region_sud': np.random.randint(15, 50, 12),
    'region_est': np.random.randint(10, 40, 12)
})

# Cr√©ation du dashboard avec subplots
fig = make_subplots(
    rows=2, cols=2,
    subplot_titles=('√âvolution des ventes mensuelles',
                    'Ventes vs Objectif',
                    'Distribution des ventes',
                    'R√©partition par r√©gion'),
    specs=[[{'type': 'scatter'}, {'type': 'bar'}],
           [{'type': 'histogram'}, {'type': 'pie'}]]
)

# 1. Ligne : √âvolution des ventes
fig.add_trace(
    go.Scatter(x=df['mois'], y=df['ventes'],
               mode='lines+markers',
               name='Ventes r√©elles',
               line=dict(color='blue', width=3)),
    row=1, col=1
)
fig.add_trace(
    go.Scatter(x=df['mois'], y=df['objectif'],
               mode='lines',
               name='Objectif',
               line=dict(color='red', dash='dash')),
    row=1, col=1
)

# 2. Barres : Comparaison ventes/objectif
colors = ['green' if v >= 100 else 'orange' for v in df['ventes']]
fig.add_trace(
    go.Bar(x=df['mois'], y=df['ventes'],
           name='Ventes',
           marker_color=colors),
    row=1, col=2
)

# 3. Histogramme : Distribution
fig.add_trace(
    go.Histogram(x=df['ventes'],
                 nbinsx=8,
                 name='Distribution',
                 marker_color='purple'),
    row=2, col=1
)

# 4. Pie : R√©partition par r√©gion
regions = ['Nord', 'Sud', 'Est']
ventes_regions = [df['region_nord'].sum(),
                  df['region_sud'].sum(),
                  df['region_est'].sum()]
fig.add_trace(
    go.Pie(labels=regions, values=ventes_regions,
           hole=0.3),
    row=2, col=2
)

# Mise √† jour du layout
fig.update_layout(
    height=800,
    showlegend=True,
    title_text='Dashboard Commercial 2024',
    title_font_size=20
)

# Personnalisation des axes
```
